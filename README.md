# 📊 Customer Shopping Behavior Analysis
# 🚀 Project Overview

This project focuses on analyzing customer shopping behavior using SQL and data analytics techniques to extract meaningful insights from transactional data.

The goal is to understand:
Customer purchasing patterns
Revenue distribution
Product performance
Customer segmentation
Impact of discounts and subscriptions

# 📁 Dataset

The dataset contains customer transaction data with features such as:
Customer ID
Gender
Age Group
Item Purchased
Category
Purchase Amount
Review Rating
Discount Applied
Shipping Type
Subscription Status
Previous Purchases
🛠️ Tools & Technologies
SQL (MySQL)
Data Analysis Techniques
Aggregations & Window Functions
(Optional) Python / Jupyter Notebook
